## Garment

## Context
An event or situation influencing outfit selection.

## Scoring Engine
A component that evaluates compatibility between garments.

## Fuzzy Logic
A rule-based reasoning method handling uncertainty using membership functions.

## Reinforcement Learning (RL)
A feedback-driven adjustment mechanism where user responses influence future recommendations.

## Wardrobe
The stored collection of available garments for a user.

## Use Case
An application-layer operation representing a system capability.
